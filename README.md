# Meitei Mayek (Kanglei) Keyboard for Windows

A custom keyboard layout for typing Meitei Mayek (Manipuri) on Windows 10/11.

## üöÄ How to Install
1. Click on **Releases** on the right side of this page.
2. Download the `Meitei_Mayek_Keyboard_v1.0.zip` file.
3. Extract (unzip) the folder to your computer.
4. Run `setup.exe`. 
   * *Note: If Windows shows a "Windows protected your PC" message, click "More info" and then "Run anyway".*

## ‚öôÔ∏èHow to Switch to Meitei Mayek
Once installed, you can switch between English and Meitei Mayek using either of these two methods:

1. **The Taskbar:** Click the language abbreviation (e.g., `ENG IN`) located next to the Wi-Fi and Volume icons in the bottom-right corner of your screen. Select **Kanglei unicode keyboard** from the list.
2. **Keyboard Shortcut:** Press **Windows Key + Spacebar** to quickly cycle through your installed keyboards.
